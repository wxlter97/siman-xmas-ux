# Siman X-Mas UX Test

Este proyecto es una aplicación web de comercio electrónico para Almacenes Siman, desarrollada con React y TailwindCSS. Incluye una barra de navegación (`NavBar`), un pie de página (`Footer`), un carrusel de categorías (`CategoryCarousel`), y una cuadrícula de secciones (`SectionGrid`). El diseño es responsivo y está optimizado para dispositivos móviles. Para la consulta de productos se utiliza la API de FakeStoreAPI.

## Instalación

1. Clona el repositorio:
    ```bash
    git clone https://github.com/wxlter97/siman-xmas-ux.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd siman-xmas-ux
    ```
3. Instala las dependencias:
    ```bash
    npm install
    ```

## Uso

1. Inicia la aplicación en modo de desarrollo:
    ```bash
    npm start
    ```
2. Abre tu navegador y ve a `http://localhost:3000`.
